# 🍽️ Swiggy Sales Analysis

A complete **Exploratory Data Analysis (EDA)** project on **Swiggy food delivery sales data** using **Python, Pandas, Matplotlib, Seaborn, and Plotly**.

This project focuses on analyzing food delivery orders to uncover **sales trends, customer behavior, food preferences, and regional performance** through KPIs and interactive visualizations.

---

## 📖 Project Overview

The goal of this project is to perform a detailed analysis of Swiggy order data and generate actionable business insights such as:

- Overall revenue generated
- Average customer rating
- Average order value
- Total orders placed
- Monthly and daily sales trends
- Revenue contribution by Veg vs Non-Veg dishes
- State-wise sales performance

This analysis can help identify:
- Peak sales periods
- Best-performing regions
- Customer ordering patterns
- Popular food preferences

---

## 🎯 Objectives

- Clean and preprocess the raw sales dataset
- Calculate important business KPIs
- Analyze time-based sales trends
- Compare food category performance (Veg vs Non-Veg)
- Identify top-performing states/cities
- Build clear and insightful visualizations for decision-making

---

## 📂 Dataset Information

The dataset contains food delivery order details such as:

- **Order Date**
- **Dish Name**
- **Price (INR)**
- **Rating**
- **Rating Count**
- **State**
- **City**

> **Note:** The dataset file used in this project is `swiggy_data.xlsx`.

## 📌 Key Performance Indicators (KPIs)

The project calculates the following KPIs:

- **Total Sales**
- **Average Rating**
- **Average Order Value**
- **Total Rating Count**
- **Total Orders**

---

## 📊 Visualizations Included

### 1. Monthly Sales Trend
Tracks revenue month-by-month to identify growth and seasonal fluctuations.

### 2. Daily Sales Trend
Shows sales distribution across weekdays to identify the strongest performing days.

### 3. Revenue Contribution: Veg vs Non-Veg
Classifies dishes based on keywords and compares revenue contribution between:
- **Veg**
- **Non-Veg**

### 4. Total Sales by State
Highlights state-wise revenue contribution to identify top-performing regions.

---

## 🧠 Insights You Can Derive

- Which months generate the highest sales
- Which weekdays perform best
- Whether **Veg** or **Non-Veg** contributes more revenue
- Which states bring in the most orders/revenue
- Customer rating patterns and satisfaction levels
